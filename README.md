# Fresher

Fresher is a minimal yet elegant full-stack web application for ordering fresh foods and cereals in Kenya.

## Features
- Product catalog with Kenyan Shilling (KES) pricing.
- Customer registration and login.
- Kenyan phone number validation (`07...`, `01...`, or `+254...`).
- Authenticated order placement only.
- User order history.
- Lightweight JSON-based persistence.

## Run locally
```bash
npm install
npm start
```
Then open `http://localhost:3000`.

## Test
```bash
npm test
```
